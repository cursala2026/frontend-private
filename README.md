# Cursala - Interfaz de Usuario (Frontend)

Este proyecto representa la cara visible del ecosistema **Cursala**. Es una aplicación web moderna diseñada para proporcionar una experiencia de aprendizaje fluida, intuitiva y profesional.

## Interacción con el Ecosistema

El frontend consume los servicios proporcionados por el [Backend-Private](../backend-private/) para permitir a los usuarios interactuar con los contenidos, gestionar sus itinerarios formativos y visualizar sus logros.

## Funcionalidades del Cliente

*   **Experiencia del Alumno:** Dashboard de cursos, visor de clases multimedia y sistema de cuestionarios interactivo.
*   **Gestión de Tutores:** Herramientas para la evaluación manual de tareas y seguimiento de grupos.
*   **Administración:** Paneles de control para la gestión de contenido y configuración de la plataforma.
*   **Adaptabilidad:** Diseño responsivo optimizado para diversos dispositivos y entornos de estudio.

## Stack Tecnológico

*   **Framework:** Angular.
*   **Estilos:** Tailwind CSS.
*   **Validación y Calidad:** E2E Testing con Playwright y Unit Testing con Jest.
